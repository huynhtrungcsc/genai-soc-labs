# Security Policy

## Supported Scope

This repository is a defensive AI cybersecurity lab portfolio. The intended use cases are SOC operations, threat hunting, incident response, compliance support, and authorized security engineering.

## Reporting a Vulnerability

If you discover a vulnerability in this repository, please open a private GitHub security advisory or contact the maintainer through GitHub. Do not publish working exploit details publicly before the issue is reviewed.

Please include:

- Affected lab or component
- Impact and affected workflow
- Reproduction steps using synthetic data when possible
- Suggested remediation if available

## Responsible Use Boundaries

Contributions and issues must not include:

- Malware source code or malware deployment instructions
- Credential theft workflows
- Unauthorized scanning or exploitation guidance
- Real secrets, tokens, access keys, or private logs
- Data from third parties without permission

Security content should be written for detection, analysis, hardening, response, and authorized validation.
