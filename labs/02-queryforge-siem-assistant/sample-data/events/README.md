# Synthetic Security Events

This dataset is fictional and safe to publish.

It contains normalized SOC events for:

- VPN authentication failures
- Finance department outbound traffic to foreign destinations
- Suspicious PowerShell
- SMB administrative share access
- C2-like beaconing
- Benign HR and IT events for contrast

The local execution engine uses this file to demonstrate end-to-end query generation and validation without connecting to a real SIEM.
