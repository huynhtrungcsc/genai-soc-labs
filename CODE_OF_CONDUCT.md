# Code of Conduct

This project is a professional learning and portfolio space. Be respectful, precise, and constructive.

## Expected Behavior

- Discuss security topics from a defensive and authorized perspective.
- Give feedback that improves clarity, correctness, and safety.
- Respect privacy and avoid sharing sensitive data.
- Keep discussions technical and professional.

## Unacceptable Behavior

- Harassment or personal attacks
- Sharing secrets, credentials, or private logs
- Encouraging unauthorized access or harmful activity
- Submitting offensive security material outside approved defensive framing
